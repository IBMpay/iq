產品 / 一般

一般標籤

在一般底下有以下基本資訊需要填寫：

Product Name * 填寫貨品的名字
Description 填寫貨品的介紹
Meta Tag Title * 填寫貨品於搜尋器顯示的標題
Meta Tag Description 填寫貨品於搜尋器顯示的介紹
Meta Tag Keywords 填寫貨品於搜尋器可檢索的關鍵字
Product Tags 填寫貨品的標籤，會於貨品頁面內顯示，點擊後可以顯示相關貨品
* 為必填項目
