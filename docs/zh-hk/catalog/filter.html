篩選

篩選功能於產品類別及產品中的 資料(Data) 頁面中都會看見相關的欄位，但即使怎樣按下去還沒有反應，原因是這個功能還未有任何選項在內。

當進入篩選清單內，見上圖，立即創建新的篩選項目再跳進下一個頁面。

到加創建頁面後，現在可以先試建立第一個篩選資訊，但要留意下列的欄位：

篩選類別 (Filter Group)

Filter Group Name * 篩選類別的名字
Sort Order 群組次序，這個次序會影響模組於前台時出現時的次序，並不是管理平台清單上所看見的次序。如沒有特定次序可填 0 或留空此欄。
* 為必填項目

篩選項目 (Filter Values)

Filter Name * 篩選項目的名字
Sort Order 篩選項目次序，這個次序會影響模組於前台時出現時的次序，並不是管理平台清單上所看見的次序。如沒有特定次序可填 0 或留空此欄。
* 為必填項目

前往產品類別頁面

假設現在我們在 Desktop 的類別中進行編輯，在 資料(Data) 標籤中找到 Filters 並將兩個顏色項目都加到這個類別內然後儲存修改。

前往產品頁面

點擊貨品中 資料(Data) 標籤，於 Filters 的欄位中輸入相關的篩選項目，點選「Red」並儲存。

完成後，在啟用了篩選功能模組的狀態下便可以看到該產品類別會有相關項目出現，有關如何啟用模組到店面，詳情請參閱頁面設計。
