重複性配置

配置項目是從 OpenCart 版本 1.5.6 就加入的特色元素之一。建立此項目時可以使顧客能夠購買產品從結賬時選擇分期付款。此功能讓使用者更能輕易地收取顧客如何收取產品或服務的費用。

前往 產品目錄(Catalog) 第三個選項便可以進入 重複性配置 (Recurring Profiles)，新增或修改現有配置文件可選清單頁面進行。

創建定期文件

Name * 分期項目
* 為必填項目

Recurring Profile

Price 每期所支付的金額
Duration 維持時間 / 期數
Cycle 多少天/週/月/年執行付費
Frequency 對應上一項 Cycle 決定單位，共有五個選項：
  • Day - 天
  • Week - 週
  • Semi Month - 半個月
  • Month - 月
  • Year - 年
Status 設定此項目可否於產品中選擇
Sort Order 於付款時顯示的排序

在下方的 Trial Profile 是支付平台提供一個伺服器給你測試，基本上跟上述都相同。

在該示例中，試用周期和試用持續時間都被設置為 1 並且頻率被設置為 月。 創建一個只有一個月的免費試用計劃。 如果想進行持續兩週的免費試用，則可以將試用期限設置為 1 。試用周期到 2 並將試用頻率設置為 週。

選取重複性配置到產品中

完成後將檔案儲存，新的文件將會顯示在清單上。接下來前往產品頁面，在 Recurring 標籤 中新增剛創建的文件加到產品中並選取適用的客戶群組，當完成更改後，產品的配置經已完成並儲存。有關如何設定客戶群組，可參閱客戶群組。
