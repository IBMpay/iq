資訊頁面

資訊頁面預設在網站的頁尾位置，而在系統的設定下，範例顯示了公司信息，見下圖。

資訊頁面功能位於 產品目錄 (Catalog) 中的最尾一項，在這個部份，使用者可以修改、刪除者新增資訊頁面。

About Us, Delivery Information, Privacy Policy, and Terms & Conditions 是預設的資料，因為對一家公司而言，這些資訊是必須的。

當進入編輯頁面後，可以看見以下四個標籤的內容：

一般 (General)

Information Title * 資訊頁面的標題
Description * 資訊頁面內容
Meta Tag Title * 填寫資訊頁面於搜尋器顯示的標題
Meta Tag Description 填寫資訊頁面於搜尋器顯示的介紹
Meta Tag Keywords 填寫資訊頁面於搜尋器中的關鍵字
* 為必填項目

資料 (Data)

Stores 選擇資訊頁面適用於哪個網店，只適用於多重網店功能
Bottom 點選後便可以從前台的頁尾Information分類中看見。
Status 設定資訊頁面會否顯示於網店內
Sort Order 設定資訊頁面在前台的出現次序

SEO

SEO是由OpenCart 3.0開始獨立抽出來，2.X的版本是放置在DATA內。當系統內的SEO URL選項啟動後，網站內凡有設定SEO URL的貨品類別都會自動轉成所填寫的項目。

資料 (Data)

這部份是可以選取這個貨品類別與指定頁面有一樣的模組會於前台顯示，如果沒有特定的選項留空白就可以。
