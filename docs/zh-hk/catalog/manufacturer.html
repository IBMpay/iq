品牌

此功能可以將產品以同一品牌的情況下列出，也是另一種篩選功能之一。從產品目錄中往下數第七項就可以前往品牌的清單頁面。

OpenCart 系統經已有了六個示範資料在內：Apple、Canon、Hewlett-Packard、HTC、Palm及Sony。如果這些品牌並不適用，使用者可自行點選刪除。在建立產品之，使用者可以考慮先創建品牌，雖然系統沒有規定每件產品都必須有品牌資訊，但若果後期整理品牌資訊必定比前期預先準備較煩瑣；而沒有標上品牌的產品是不會於任何品牌面中出現。

建立新品牌便可以點擊右上方的新增按鈕，進入下一個頁面後便要填寫以下幾項：

Gerneal (一般)

Manufacturer Name * 品牌名字
Stores 設定於哪一間OpenCart商店中使用，如沒有多重OpenCart商店就不用理會。
Image 可上載品牌的商標，有關如何上載圖片可參閱圖片管理。
Sort Order 群組次序，這個次序會影響品牌於前台時出現時的次序，並不是管理平台清單上所看見的次序。如沒有特定次序可填 0 或留空此欄。
* 為必填項目


SEO

這部份與貨品及類別相同，填寫後，如果系統使用SEO URL功能就可以利用簡短的路徑進入該頁。

如需要修改現有的品牌資訊，於清單面頁按下項目最右側的編輯按鈕（見鉛筆圖示），並依據上述的內容進行修改就可以。

即便使用者之後將品牌資訊修改了，但都毋需重新為相關產品再修改品牌訊息。因為OpenCart會因應作出的改動而自動將訊息更新；另外，若果要刪除品牌資訊，使用者必須將所有相闗產品都品牌都除去或是刪除相關的所有產品才能刪除該品牌。

建立新的品牌資訊

為了提升商店的印象，品牌是一家商店不能缺少的元素，所有產品最好都有相闗品牌會較容易使顧客留下印象。有關如何設定品牌於產品內，詳情請參閱管理者介面中產品頁面。為了使用者便具體了解品牌的操作，以下步驟將會示範如何創鍵新的品牌於 OpenCart 系統內。

創建新的品牌 Acer 於 OpenCart 系統中，由於暫時只有一個商店，所以預設已點選了 Default，如有圖片及排序資料便可以自行輸入，然後跳到 SEO標籤 中填寫 keyword 為 acer。

接著前往產品頁面，建立一個新產品命名 Acer Aspire 並將其品牌設定為 Acer，填寫好所需的資料後便可以儲存新產品。

前往店面的品牌頁面

接下來到店面的頁尾位置，於 Extras 的欄位第一項 Brands 可以進入所有品牌的頁面。

當顧客前往品牌頁面後，便可以立即發現所有品牌並已首一個字母順序排列，而 Acer 剛好就在 A 行的第一個。

點擊了 Acer 後，系統便會抽出所有屬於該品牌的產品並列出來，如下圖中，只有剛剛建立的 Acer Aspire 會顯示在頁面內。

點擊 Acer Aspire 後會連接到該產品頁面，顧客亦可從頁面內，見藍色框的位置，品牌亦會從這裡顯示出來，可以讓顧客更容易尋找該品牌於店內的其他產品。

Carousel 模組

在系統預設的 Carousel 中是放置了品牌的商標並由模組和橫額群運算而生成，但新增了新的品牌後是不會自動產生到 Carousel 當中，原因是新的品牌資訊需要再進一步到橫額群內編輯。使用者必須將品牌圖片加到該群組內再儲存才能於店面顯示出來。

有關新增及設置詳情，可參閱系統目錄中的設計的横額及頁面設計。
