產品類別概覽

產品類別

於預設主題而言，主要類別(Parent Categories)會於網頁的頂部以及產品頁面的左側空間顯示出來。類別功能主要是將類同的產品細分，這好讓顧客可以更容易找到他們所需的東西。同時間店內的產品做好類別後，顧客即使要作出資訊及數據上的比較時也可以容易看出差別而更快速作出決定。

每當新增一款產品時，使用者都必須為產品載入其類別，所以必須先建立了產品類別才能加入產品到店內，這樣的流程可以減少產品資料變得混亂以及減少不必要的重覆編輯動作。

OpenCart商店中的產品都必須點選所屬的類別，若果沒有被類別，產品會無法於前台任何類別中顯示。除非於產品搜索中輸入關鍵字或從品牌選項中尋找。

在管理者介面中點擊左側目錄 (Catalog)展開次目錄，進入產品類別的清單頁面可以看見每項類別名稱，如下圖。

圖中的類別資料於安裝時經已預設載入商店中，使用者可以點選類別旁邊的空格或表格頂部的空格把這頁面的全部項目刪除。剔取了所刪除項目後，點擊右上方的紅色按鈕後便可以完成刪除程序。

> 符號會出現於類別與類別之間，這代表了後者是前者的次類別。例子：
Laptops & Notebooks > Windows
在這裡 Windows 是 Laptops & Notebooks 的次類別，所以在店面中會看見 Windows 會顯示在 Laptops & Notebooks 的下拉式表單中。

排序(Sort Order)功能主要負責將產品類別按使用者的意向順序顯示在產品網頁頂部及產品頁面左側的清單中。

若填寫資料時把排序欄留空，系統會自動設置為 0 並自動排列

使用者想編輯現有的產品類別時，只要點擊清單中最右側的藍色按鈕就可以進入編輯頁面。

創建新產品類別

以上大致講述了產品類別的特性，以下的部份將會介紹如果創建新的產品類別。產品類別是 OpenCart 系統中的特色之一，整個系統基本上是圍繞這些產品類別而構成。每個產品類別在商店中都擁有獨立一頁來顯示其下類別的產品。

使用者於開始加入產品前，當前先要決定如何將產品類別。一般來說，需要從創建一個廣泛的主要類別開始，然後將其縮小到每個次類別。這樣可以提高商店的可用性，從而將顧客帶到商店中他們亦可能感興趣的產品。

前往產品類別清單頁面，若要創建一個全新的類別，只要點擊於面頁右上方的藍色按鈕，系統就會跳至創建畫面。進入新的畫面後便會看到四個主要填寫的類別：General、Data、SEO、Design
