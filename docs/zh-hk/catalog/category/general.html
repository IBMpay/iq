產品類別 / 一般

一般標籤

在一般底下有以下基本資訊需要填寫：

Category Name * 填寫貨品類別的名字
Description 填寫貨品類別的介紹
Meta Tag Title * 填寫貨品類別於搜尋器顯示的標題
Meta Tag Description 填寫貨品類別於搜尋器顯示的介紹
Meta Tag Keywords 填寫貨品類別的關鍵字，客戶於搜尋器中輸入相對關鍵字以便搜尋
* 為必填項目
