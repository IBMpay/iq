產品類別 / 資料

資料標籤

此標籤需填寫以下資訊：

Parent 選擇產品類別的主要產品類別
Filters 選擇貨品類別可以搜尋的特徵
Stores 選擇貨品類別適用於哪個網店，只適用於多重網店功能
Image 貨品類別的大圖片，可空白，可以上載到網站再點選
Top 點選後便可以從前台的選單中看見，只適用於主要產品類別
Columns 設定這個貨品類別的子類分多了個欄表示
Sort Order 設定貨品類別在前台的出現次序
Status 設定貨品類別會否顯示於網店內
