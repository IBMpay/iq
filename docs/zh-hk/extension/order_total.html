帳單金額

使用者可於此部份設定產品以外的額外收費，此部份可於插件頁下拉式表單內找到。額外收費可以包含：運費、手續費、稅項等這類收費。有關訂單總數的更多信息，請參閱訂單。OpenCart系統經已預設下列帳單金額設定項目。

上圖顯示了部份已啟用的功能，若不需要其功能，使用者可以於   編輯  內選擇「不啟用」或點擊   卸載  即可

Coupon

優惠券，當狀態為「啟用」時，此功能將會添加到訂單總計頁面。排序即是順序影響擴展插入的順序。顧客亦可以於結帳前使用相關的優惠編號於店內使用。

Store credit

商店點數，設定「啟用」後顧客便可以於購物車的頁面中看見是次消費會自動扣除的點數，有關點數的情詳可參閱用戶概覽。

Handling fee

手續費，設定訂單於超出指定金額會收取額外費用。

Order Total 訂單金額
Fee 超出以上之金額將會收取之費用
Tax Class 稅項設定
Status 「啟用」或「不啟用」此項目
Sort Order 項目顯示的次序

Klarna Fee

添加Klarna費用包括以下信息，只適用於支付平台的 Klarna Invoice 選項：

Order Total 訂單金額
Invoice Fee -
Tax Class 稅項設定
Status 「啟用」或「不啟用」此項目
Sort Order 項目顯示的次序

Low order fee

設定每張訂單的最低消費金額，若不乎合定額便會收取指定的額外費用。

Order Total 訂單金額
Fee 少於以上之金額將會收取之費用
Tax Class 稅項設定
Status 「啟用」或「不啟用」此項目
Sort Order 項目顯示的次序

Reward points

積分點數換領，設定「啟用」後顧客便可以於購物車的頁面中看見是次消費會自動扣除的點數，有關積分換領的情詳可參閱產品積分。

Shipping

運輸方法，設定「啟用」後系統將自動估算運費並將費用添加到訂單總額中。

Sub-total

小計，設定「啟用」後顧客便可以於購物車的頁面中看見是次消費的所有產品的合共金額，不包括運費。Select the status and sort order of the sub-totals in checkout.

Taxes

稅項，設定「啟用」後顧客便可以於購物車的頁面中看見是次消費所需支付的總稅項征費。

Total

消費總金額，設定「啟用」後顧客便可以於購物車的頁面中看見是次消費所需支付的總金額。

Gift-voucher

禮品券，設定「啟用」後顧客便可以於購物車的頁面中使用相關的禮品券。
