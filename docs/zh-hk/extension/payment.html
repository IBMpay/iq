編輯付款方式

在管理者介面內，啟用各個支付平台（如Paypal）都必須要先創建相關帳戶。在編輯的頁面上有著不同的資料欄供使用者填寫，內裡的資訊都可以從該支付平台中獲取，填寫正確的資料後便可以輕輕鬆鬆在商店完成交易處理。以下為共通需要填寫的資料，使其付款方式可以於商店中顯示選項讓顧客選擇：

Total 設定購物金額到指定數目才可以供顧客選用。
Order Status 當顧客確認訂單時，設定訂單狀態，有關修改訂單狀態的詳情請參閱系統 > 本地化中的訂單狀態。
Status 「啟用」或「不啟用」該付款方式。
Sort Order 顯示該付款方式的次序。

結帳頁面

在此頁中的第5步，顧客需要在完成購買前選擇一種付款方式。只要合乎指定金額、地區等條件，顧客便可以選擇使用者已啟用的支付平台來完成購物程序。

交易紀錄

完成購物後，顧客只要前往「我的賬戶」(My Account)中的「交易紀錄」(Order History)，如上圖所示可以看見最新的訂單狀態為「Pending」。因為設定將支付方式的訂單狀態預設為「Pending」。

只要使用者檢查了交易完成或失敗，使用者可以前往管理者介面中的「銷售」(Sales)的「訂單」(Order)中編輯該訂單的狀態，詳情請參閱訂單。

技術支援

如需獲得對部份付款方式(插件)的技術支援，請訪問官方論壇的付款方式擴展部分。
