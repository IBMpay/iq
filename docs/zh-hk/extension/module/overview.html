核心模組

概覽

最新版本的OpenCart提供了26個免費核心模塊：

Account 啟用後可以於版面設計中放置用戶連結在指定頁面及位置上。
Banner 添加圖片橫額，圖標鏈接到商店中的不同產品或製造商。
Bestsellers 啟用後可以於版面設計中放置最熱賣產品在指定頁面及位置上。系統會按照銷售情況而自動排列產品次序。
Carousel 顯示可作為幻燈片瀏覽的橫額。
Category 啟用後可以於版面設計中放置產品類別在指定頁面及位置上。
Divido Product Page Calculator
Featured 建立全新的版塊，選擇指定精選貨品並於版面設計中放置在指定頁面及位置上。
Filter 啟用後可以於版面設計中放置籂選功能在指定頁面及位置上。
Google Hangouts A chat box used with a Google account to interact with customers.
HTML Content 建立自定HTML內容並於版面設計中放置此版塊在指定頁面及位置上。
Information 啟用後可以於版面設計中放置資訊清單在指定頁面及位置上。
Klarna Checkout
Latest 啟用後可以於版面設計中放置最新產品在指定頁面及位置上。系統會按照產品更新的時間而自動排列產品次序。
Lay-Buy Layout
Log In with PayPal
Login with Amazon
Pay with Amazon
PayPal (Powered by Braintree) Button
PayPal Express Checkout Button
Pilibaba Checkout Button
Sagepay Direct Card Management
Sagepay Server Card Management
Slideshow Displays banner that can be navigated through as a slideshow.
Specials 啟用後可以於版面設計中放置特價產品在指定頁面及位置上。系統會按照產品更新的時間而自動排列產品次序。詳情可參閱特價產品。
Store 啟用後可以於版面設計中放置連結到其他網店(Muilt Store)的目錄在指定頁面及位置上，更可設定只限管理者人員才顯然此功能。
eBay Listing

所有模組都可以於設計版面中進行修改位置、狀態和排序及順序進行操作。
