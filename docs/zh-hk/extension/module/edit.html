編輯模組

模組在用途和功能上可能會有很大差異，根據不同模組的不同，定制適合特定商店的模組也會有所不同。模組修改頁面內有四個常見項目：

Layout Position
Status Sort Order

Layout - 排版

排版可以決定模組於哪一頁中，在甚麼位置顯示出來。如下圖所示，進入「設計」(Design)選單的「排版」(Layout)後可以看到以下內容，系統於安裝時預設了的13頁不同類型的頁排版。如果模組須要在多個版面中呈現出來，那麼使用者必須要在每一個頁面中進行編輯。

下圖展示了，將一個精選項目加到「產品分類」的排版的左側中，所以只要顧客進入任何一頁的的「產品分類」中都會看見這個模組所呈現的內容。但核心模組可以呈顯在部份排版頁面中，但不能分配單獨的頁面。

Position - 位置

排版位置選擇決定模組在頁面上的顯示位置。在排版內選擇的頁面類型中，模組可放置位於頁面的頂部、底部、左側或右側。

假如將精選項目從左側移頂部，效果便會變成如下圖所示。

精選項目的框在產品類別頁面的產品上方列出，從本來其左側的位置移開。

Status - 模組狀態

「啟用」或「不啟用」模組的狀態將影響模組在商店前台上的可見性。「不啟用」模組與「卸載」模組不同，「不啟用」只是將其內容不顯示出來，「卸載」則是把已輸入的資料刪除，若再次使用時也必須重新填寫所有資料。

Sort order - 排序

排序。假設模組 Feature與模組 Specials兩者都放置在底部的位置時，系統會按模組的名字優先排序。結果就會得到模組 Feature會先在底部位置顯示，下方接著是模組 Specials。
