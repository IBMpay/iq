安裝 / 卸載模組

OpenCart系統預設了數個核心模組，只要前往插件模組清單中便可以進行安裝及卸載模組的程序。預設的模組中並不是全部都自動安裝，使用者可以嘗試自行安裝並啟用不同的模組，這樣一來可以更熟悉OpenCart的操作，二來也可以了解哪些功能較合適用於商店。

安裝

如圖中所示，凡是未進行安裝的模組都會顯示   安裝  ，只要安裝後就會顯示為以下的畫面。

卸載

相反，凡是已安裝的模組都會顯示   卸載  ；只要按下此按鈕，所有已設定的資料都將會移除。

卸載的模組再使用時需要重新安裝，該模組內的資料在卸載後無法恢復，所有資料都必須重新填寫。
