遷移OpenCart至新伺服器

假如要轉換伺服器，OpenCart的系統可以很容易就將其檔案及資料備份，只要按著以下步驟就可以輕鬆完成資料轉移：

  1. 從現有的伺服器中，透過 FTP 工具 或 cPanel 下載所以 OpenCart的檔案，當中包括已安裝的插件、上載的產品圖片、config.php 等檔案。當擁有一個完整的文件備份後，便可以上載到新的伺服器中。
  2. 備份資料庫所有資料：
    1. 從 OpenCart 管理者平台匯出資料
      登入現有的伺服器 OpenCart 管理者平台，前往 備份及重置 (Backup & Restore) 將所有資料從頁中 匯出。
    2. 經由cPanel進入資料庫後台
      尋找安裝了 OpenCart 的資料庫，直接將整個資料庫匯出並下載到電腦中作備份。
  3. 從備份了的文件中，編輯 config.php 及 admin/config.php (建議複製此文件作備份之用)，將文件內的資料庫資訊及OpenCart在新伺服器內的路徑重新改寫，完成後就可以上載到新伺服器內。此部份會影響到整個 OpenCart 系統的運作，如有任何疑問可以前往 OpenCart 官方論壇 或 OpenCart 技術支援查詢。
