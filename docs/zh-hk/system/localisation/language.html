語言

語言功能可以使商店迎合使用不同語言的顧客，從而令他們可以在商店消費。OpenCart預設系統只支援英語，若果要為系統增添新語言必須前往OpenCart 官方市場(Marketplace)購買及下載合適系統所使用的語言包。有關如何安裝新的語言包，請查看新增語言教學內容。

影片教學

假設已經成功安裝了一種新語言，前往語言清單的面頁，如下圖所示，點擊右上方的  新增  按鈕前往下一頁。


Language Name * 設定語言統稱於管理者介面中顯示。
Code * 國際通用的語言編碼(ISO language code)，如不清楚新增的語言編碼，可前往W3Schools查詢完整清單。
Locale * 瀏覽器用來自動檢測網站語言的代碼。 區域代碼可以查看這裡。
Status 啟用或隱藏商店的語言選項
Sort Order 語言顯示排序
