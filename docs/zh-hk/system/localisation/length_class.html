長度單位

設定長度單位的作用是使建立產品時，可以精確將產品資料呈現在網頁中。更重要是，按照系統的計算，在送貨選項的會自動調整運費。

建立後的長度單位可以於產品中的資料標籤內作出選擇，添加到產品的數值也將顯示在商店前面的產品頁面中。下圖是建立單位所需要的填寫項目：

Length title * 長度名稱
Length unit * 長度單位
Value 預設的數值填寫為1.0000，其他的轉換數值則以此倍大或減少
* 為必填項目
名稱 厘米(預設) 米 毫米
單位 cm m mm
數值 1.0000 0.0100 10.0000
