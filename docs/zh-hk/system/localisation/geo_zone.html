地理區域

地理區域是用於計算運費和稅費的自定義區域。 多個國家和地區可以添加到一個地理區域來自定義運輸方式。

下圖的顯示如何設置一個新的地理區域，填寫定義的名稱和簡介後，按下  新增  來添加此項目中所受理的地域。

Geo Zone Name * 地理區域名稱
Description * 簡介
* 為必填項目

Country 受理這個地理區域的國家
Zone 於所選國家的地區

設置完成地理區域後可以前往運輸平台了解如何設定不同的地理區域。
