退貨

退貨功能允許使用者建立不同的退貨退款狀態，系統會依照三個部份在不同的位置呈現出來。

影片教學

影片教學由OpenCart Developer, iSenseLabs提供

退貨狀態

Return Status Name * 退貨狀態名稱
* 為必填項目

退貨動作

Return Action Name * 退貨動作名稱
* 為必填項目

退貨理由

Return Reason Name * 退貨理由名稱
* 為必填項目

退貨理由設置後可以在店面的退貨頁中查看，退貨頁面在網站最底部 Customer Service 分類內。
