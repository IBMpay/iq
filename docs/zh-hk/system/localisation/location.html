實體店(分店)資訊

假若使用者擁有一間或多間的實體店鋪，可以從這個頁面功能增設店舖資訊。這樣一來，將實體店與網上商店連結在一起後使推動營稍會有更大的幫助。

使用者可以從左側的選單展開 系統(System) 選單，再尋找 本地化(Localisation) 中的 實體店(分店)資訊 (Store Location) 點擊並進入。

如上圖所示，系統預設是沒有任何資訊是給予這個部份的，所以點擊右上方的  新增  按鈕前往下一頁。

Store Name * 店舖名字
Address * 店舖地址
Geocode 填寫 Google Map 的經緯線度數
Telephone * 店舖的聯絡號碼
Fax 店舖的傳真號碼
Image 店舖的外觀，此項目會於聯絡我們頁面中出現。
Opening Times 營業時間
Comment 店舖簡介

完成店舖的資訊並儲存後，再前往 系統(Setting) 頁面的 商店設定(Store Tag) 中，如下圖所示，頁面的底部會有一個新的項目顯示出來，只要點選新建立的店輔資訊，這樣就可以把剛建立的項目從店面顯示出來。
