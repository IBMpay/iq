重量單位

與長度單位相同，建立後的重量單位可以於產品中的資料標籤內作出選擇，添加到產品的數值也將顯示在商店前面的產品頁面中。下圖是建立單位所需要的填寫項目：

Weight title * 重量名稱
Weight unit * 重量單位
Value 預設的數值填寫為1.0000，其他的轉換數值則以此倍大或減少
* 為必填項目
名稱 公斤(預設) 克 磅
單位 kg g lb
數值 1.0000 1000.0000 2.2046
