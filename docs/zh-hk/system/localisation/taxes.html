稅項設定

本地化中的稅收部分允許使用者創建將應用於商店中特定產品的稅款。

稅率

稅率將用於計算訂單中的稅金。 需要以下信息來為主管部門節省稅率:

Tax Name * 稅項名稱
Tax Rate * 稅率
Type 可以設定此稅項是由百份率計算還是以實數作為計算。
Customer Group 設定此稅率適用於指定的用戶群組
Geo Zone 設定適用的地理區域的範圍。
* 為必填項目

稅項類別

稅務類別可以將多個稅率存儲到一個類別中。 可以根據付款地址（客戶）或商店地址自定義稅額來計算稅金。 稅務類別需要以下信息：

Tax Class Title * 重量名稱
Description * 重量單位
Tax Rate 基於需要選擇付款地址或商店地址。 這將確定添加到店面購物車總額中的稅額。 如果其他稅率列在其中，則優先級決定稅率的位置。
* 為必填項目

稅單必須單獨添加到每個產品，以便為購買申請稅率。 稅類字段位於產品的數據選項卡下。 查看產品獲取更多信息.
