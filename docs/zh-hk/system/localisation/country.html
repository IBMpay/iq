國家

在預設的資料中，OpenCart 經已包含了239個國家的資料，如果沒有合適的國家或區域，使用者可以自行增設不同的國家使商店變得更大聚化。

Country Name * 國家名稱
ISO Code (2) 2位的國際編號
ISO Code (3) 3位的國際編號
Address Format 設定不同地區的郵寄格式
Postcode Required 某些國家或地區沒有郵寄編號，如沒有需要的地方選否即可。
Status 可以將國家或地區的狀態隱藏，隱藏後便不會在店面中任何位置顯示出來。
* 為必填項目
