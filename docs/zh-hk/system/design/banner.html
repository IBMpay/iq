橫額

這裡的橫額是指店面首頁中的巨大橫額，可以加上連結使其與網站內外進行連結。橫額的主要功用是可以吸引顧客的眼睛，並引導他們到設定好的頁面或產品資料中。如要編輯所有橫額項目可從管理者介面前往 設計(Design) 選單的 橫額(Banner)頁面。

模組與橫額

在系統預設中，橫額有三個經已設置了：首頁的巨大橫額幻燈片、下方的商標 Carousel 以及產品分類的牌品精選產品示範。在橫額清單中主要是管理每一個項中有哪一張圖片在同一個項目內，完成該項目後，使用者就必須前往模組頁面到 Slideshow 和 Carousel 內建立新的橫額項目，將在前者已建立的項目都新增一個模組，儲存後就可以在頁面設計中使用。

Banner Name 設定橫額的標題，方便於模組時尋找
Status 設定此項的的狀態，隱藏時此項目就無法在前台顯示
Title 滑鼠指向橫額時，會顯示此標題
Link 連結到指定頁面
Image 透過系統的圖片管理可以選擇顯示的橫額圖片
Sort Order 設定圖片的顯示次序
