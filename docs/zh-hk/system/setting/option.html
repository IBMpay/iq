選項設定

選項主要是控制網站內的顯示資訊以及商店的基本功能如評級、稅項設定、帳戶、結帳項目等。

產品

Category Product Count 設定店面於選單中，是否顯示產品總數量
Default Items Per Page (Admin) * 設定管理者介面中，產品的每頁顯示數量

店面顯示

當 Category Product Count 設定為啟用時，如下圖所示，選單內的每個產品分類都有著產品總數於頂端選單內。

如果商店內很多產品類別，強力建意使用者將此功能關閉，這樣一來可以減少網站系統的運算工作而令網店更快速顯示所有內容。

評級

Allow Reviews 允許會員摘寫貨品評級
Allow Guest Reviews 允許非會員顧客摘寫貨品評級

優惠券

Voucher Min 優惠券的最小面額
Voucher Max 優惠券的最大面額

稅項

Display Prices With Tax 啟用稅項，如果不需要可以關閉此功能
Use Store Tax Address 如果客戶未登錄，則使用商店地址計算稅金。使用者可以選擇使用商店地址作為客戶的送貨地址或付款地址。
Use Customer Tax Address 登錄時使用客戶的默認地址來計算稅金。 使用者可以選擇使用客戶送貨地址或付款地址的默認地址。

帳戶

Customers Online 通過客戶報告部分在線跟踪客戶。
Customers Activity 通過客戶報告部分跟踪客戶活動。
Log Customer Searches 記錄顧客搜尋紀綠
Customer Group 預設顧客群組
Customer Groups 顯示是否在登記時給顧客知道自己的顧客群。
Login Display Prices 設定是不會員才能看見價錢
Max Login Attempts 密碼錯誤時可允許會員嘗試登入次數，超出此數值後需等一小時系統自動解鎖或經由使用者從管理者介面手動解鎖。
Account Terms 設定登記成員時所使用的條款頁面。

結帳

Invoice Prefix 自定義發票編號格式
Display Weight on Cart Page 購物車中顯示重量
Guest Checkout 允許非會員結帳
Checkout Terms 結帳時所顯示的條款及細則，如果沒則不用選擇
Order Status 設定結帳後的訂單狀態
Processing Order Status 設置客戶的訂單在訂單開始減少和優惠券，優惠券和獎勵兌現之前必須達到的訂單狀態
Complete Order Status 設置顧客的訂單在允許訪問其可下載產品和禮品券之前必須達到的訂單狀態
Fraud Order Status 當顧客被懷疑試圖改變訂單付款細節或使用已經使用過的優惠券，禮券或積分時，設置訂單狀態
API User Default API user the admin should use

庫存

Display Stock 顯示庫存
Show Out Of Stock Warning 當產品沒有庫存時，設定產品售罄的訊息
Stock Checkout 允許顧客於沒有庫存時也可以買產品

加盟

Affiliate Group 加盟成員群組
Affiliate Requires Approval 設定每一個加盟的成員要否批准才能加入
Automatic Commission 設定每完成一個訂單後會否自動計算佣金
Affiliate Commission (%) 加盟佣金
Affiliate Terms 設定登記加盟成員時所使用的條款頁面。

退款/退貨

Return Terms 設定退貨條款，需要從資訊頁面選擇
Return Status 退貨狀況

認證

Captcha 選擇認證程式
Captcha Page 需要使用認證的頁面
