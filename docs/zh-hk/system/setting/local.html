地區設定

設定地區、貨幣等個性化資訊，使顧客可以更容易了解商店。

Country 國家
Region/State 地區
Language 商店所使用的第一語言
Administration Language 管理者介面所使用的語言
Currency 商店所使用的貨幣單位
Auto Update Currency 設定會否自行更新貨幣滙率，如不適用可選擇否
Length Class 商店所使用的長度單位
Weight Class 商店所使用的重量單位
