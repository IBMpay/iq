伺服器設定

在這一標籤頁中，分為了四個部份：一般、安全性、更新及錯誤報告。

一般

Maintenance Mode 進入維護模式後，只有使用者從管理者介面登入後才能看得見店面。
Use SEO URLs 設定網站是否啟用SEO連結，要配套伺服器與 .htaccess 的設定
Robots 添加或刪除bot用戶代理訪問共享會話
Output Compression Level The level of HTTP compresion to be used in your HTTP headers, 0-9.

安全性

Use SSL 啟用SSL功能使網站可以使用HTTPS，此功能需要向服務供應商購買服務
Allow Forgotten Password 已登記的顧客再次登入時可以用忘記密碼
Use Shared Sessions 使用共享會話
Encryption Key 輸入數字和字母的組合來加密私人信息

上載

Max File Size 可設定上載檔案的上限，建意先向服務供應商查詢最大限制
Allowed File Extensions 允許可以上載的檔案格式
Allowed File Mime Types 允許可以上載的檔案MIME類型

錯誤報告

Display Errors 顯示錯誤報告
Log Errors 啟用系統中的錯誤報告
Error Log Filename * 錯誤報告檔案名稱
