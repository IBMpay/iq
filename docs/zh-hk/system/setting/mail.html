電郵設定

要從網站中得到任何電郵通知，使用者必須填寫以下的欄位。一般郵件都會用MAIL及SMTP兩種制式，如果選擇SMTP就需要填寫第一部份尾五項的資訊。

Mail Engine 可以選擇 MAIL 或 SMTP 格式，一般伺服器及電郵服務商
Mail Paramaters 填寫所接收用的電郵地址
SMTP Host 選擇SMTP時，填寫SMTP的電郵伺服器
SMTP Username SMTP登入用戶
SMTP Password SMTP密碼
SMTP Port SMTP連接埠，因應不同的服務供應商所使用的服務器而各有不同
SMTP Timeout SMTP自動終止時間，設定網站與伺服器的連接時間，如果超時便不能發送電郵
Alert Mail 可接收到電郵的情況
Additional Alert E-mails 其他同時可收取電郵的電郵地址
