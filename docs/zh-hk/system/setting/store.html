商店設定

一般的店鋪資訊都在此標籤頁內填寫，例如電話、地址、營業時間等。

Store Name * 商店名字
Store Owner * 商店老闆
Address * 商店的位置 / 辦公室的位置
Geocode 填寫 Google Map 的經緯線度數
E-Mail * 商店的聮絡電郵
Telephone * 商店的聯絡號碼
Fax 商店的傳真號碼
Image 商店的外觀，此項目會於聯絡我們頁面中出現。
Opening Times 營業時間
Comment 商店簡介
