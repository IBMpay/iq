備份 / 重置

遺失數據是網上商店最有可能發生的事件之一。商店數據庫包含客戶和產品信息這些重要資料，一旦失去所有資料後是難以恢復，並且肯定需要大量時間才能重新輸入或建立信息。所以您必須警惕將商店數據作定期備份，如每週一次，備份下載到自己的電腦內以防萬一發生。

OpenCart 系統設有數據備份及恢復功能，登入管理者平台後，展開系統清單中的維護選單可以看見 備份 / 重置 (Backup / Restore) 選項。備份包含可供下載的 MySQL 數據庫中的所有數據表，或可以選擇所需的數據表而作出備份。點擊 匯出按鈕將可以得到數據庫的備份。

如果您需要恢復先前保存到電腦內的備份文件，則只需要點擊 匯入按鈕，從電腦中選擇所需的 SQL檔案，系統使會將數據庫重置。
